# SwarmBox Community
Welcome to the SwarmBox community!  
We welcome everyone interested in drone swarm research, development, and security to join our community.

This section serves as a hub for all community-related information, 
including guidelines for contributing to the project, 
reporting security issues, 
and understanding our code of conduct.

## Essential Documents
Before diving into the codebase, we highly recommend reviewing the following documents to ensure a smooth and collaborative experience:
- [Code of Conduct](../CODE_OF_CONDUCT)
- [Contributing Guidelines](../CONTRIBUTING)
- [Security Policy](../SECURITY)

## Get Involved
The SwarmBox project is maintained on GitHub, and we actively welcome contributions from the community!

Whether you want to report a bug, suggest a new feature, or contribute code,
feel free to use the [GitHub Issues](https://github.com/postech-compsec/swarmbox/issues) 
and [Pull Requests](https://github.com/postech-compsec/swarmbox/pulls) to engage with the project.
Please make sure to follow the guidelines outlined in the [Contributing Guidelines](../CONTRIBUTING) for a healthy community.
