# Paper Reference Scripts (FSE 2026)

The scripts in this folder are not intended for the general execution or operation of the SwarmBox framework.
They are **hard-coded reference scripts** written exclusively for the purpose of generating results and figures 
for specific Research Questions (RQs) presented in our FSE 2026 paper.

**NOTICE**
- These scripts are not designed for general-purpose use.
  They contain hard-coded variables and assume a highly specific dataset directory structure.
- If you are trying to validate or reproduce the paper's results, you do not need to execute these scripts directly.
- Instead, please download our fully packaged dataset and integrated execution environment from the [Zenodo repository](../../../README.md#Zenodo-Artifact) to conduct your evaluation.
